# NoSQL-Challenge
Module 12 NoSQL Challenge

This respository consists of a Data folder which contains NoSQL_setup_starter.ipynb, NoSQL_analysis_starter.ipynb, and a Resources folder.

## NoSQL_setup_starter.ipynb
This file contains the PyMongo script written in Jupyter Notebook to set up and update the database and prepare it for analysis.

## NoSQL_analysis_starter.ipynb
This file contains the PyMongo script written in Jupyter Notebook which analyses the database which was set up in NoSQL_setup_starter.ipynb.

## Resources
This folder contains the establishments.json file which holds the data used in the database for analysis.
